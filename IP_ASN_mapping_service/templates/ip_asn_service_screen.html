IP to ASN Mapping Service

This service runs in the background and sets the ASN value for each IP object. While the service is running, it will look up the ASN value for each incoming IP object to see if the ASN value given with the data is correct. If it is not correct, the service will set the ASN to the correct value, and will leave a comment on the IP object indicating that a correction occurred.

Click "Start/Stop Service" to start or stop the service from processing. The service will process data as long as the status is 'Running'.
Click "Rerun Service" to have the service re-analyze all data in the database.
